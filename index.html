List of class you can use. Fuckanimation.

    + animation-iteration-count_infinite CLASS

  • spinLeft CLASS

    + animation-iteration-count_infinite CLASS

  • spinRight CLASS

    + animation-iteration-count_infinite CLASS

  • animation-iteration-count_X CLASS

    Special Class use, X = Repeat Number

    animation-iteration-count_3 = repeat animation 3 times.

    animation-iteration-count_infinite = loop animation infinity.

  • animation-duration_X CLASS

    Special Class use, X = Duration of animation in seconds.

    animation-duration_3 = Time animation 3 seconds.
